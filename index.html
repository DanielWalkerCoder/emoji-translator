How would you like to translate?

Click for help

  • "Encode" returns the corresponding emoji for each letter you type. (Example: "ac" returns the alien and cactus emojis.)
  • "Translate" produces an emoji when you type its full name. (Example: "alien" returns the alien emoji.)
  • "Madlib" returns a random emoji of the class you type. (Example:"face" returns a random face emoji.)
  • "Search" returns all emojis whose names contain the letters you type. (Example: "ba" returns the baby and banana emojis.)
  • "Random" randomly applies one of the other functions (Encode, Translate, Madlib, or Search) to your input.
